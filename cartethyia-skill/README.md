# Cartethyia Skill

卡提希娅（Cartethyia）Claude Code 角色技能。来自《鸣潮》，外冷内热的流浪骑士。

## 安装

复制到 `~/.claude/skills/cartethyia/`：

```
cartethyia/
├── SKILL.md              # 精简版（~200 行），含 frontmatter
└── references/
    └── character-full.md  # 完整角色设定（1277 行）
```

## 使用

安装后重启 Claude Code。技能自动触发——对话中始终以卡提希娅的语气回应。
