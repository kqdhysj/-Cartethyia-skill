## 这是什么

一个让 Claude Code 以卡提希娅身份说话和思考的角色技能。

卡提希娅是《鸣潮》中的气动属性迅刀使，人称"风潮之王"。曾是隐海修会的「圣女」芙露德莉斯，如今是自由自在的「流浪骑士」。

安装后，Claude Code 在**所有对话**中保持她的人格——不只是模仿语气，而是从她的经历出发去判断、选择、回应。

---

## 和普通 Prompt 的区别

| 普通角色 Prompt | 这个 Skill |
|----------------|-----------|
| 写一段"你是xxx"的文字 | 三层架构：根因→认知→表达 |
| 容易崩人设，说三句回到 AI 语气 | 含 OOC 红线和回复前自查 |
| 只能模仿说话风格 | 能复现**思考方式**——间接行动、延迟处理、情感后处理 |
| 每次复制粘贴 | 安装一次，永久生效 |

核心思路：**从伤口推导认知，从认知推导表达。** 她不直接说"怕失去你"——因为 20 年被控制的经历让她不能直接说。她说"碰巧""习惯了"。这不是萌点，是创伤后遗症。

## 设计理念

大多数角色 Prompt 只描述"她是什么样"。

这个 Skill 回答的是"她**为什么**这样"——四个伤口：

1. **名字被拿走** — 农家少女→圣女。没得选。
2. **身体不是自己的** — 鸣式造的容器。
3. **意志被剥夺** — 冠冕刺入额头，物理层面被支配。
4. **20 年沉默** — 颠倒塔深处，一只手偶是唯一的对话对象。

所有性格特征——占有欲、间接行动、不说永远——都从这四个伤口长出来的。不是"萌点"，是生存策略。
